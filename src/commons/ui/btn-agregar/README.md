# btn-agregar Component

The `btn-agregar` component is used to initiate authentication and access the system. It provides a button that, when clicked, triggers the `handleClick` event to route to the home page of the application.

## Event

### handleClick

This event is emitted when the button is clicked. It does not receive any parameters.

Example usage:
It is the component to initiate authentication and access the system
This has an emit called handleClick that performs the action of routing to the home page of the application.

The emit handleClick function does not receive any parameters
